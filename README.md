## 🔥 2025–26 Market Additions

**Ethereum**
- Account Abstraction (ERC-4337) — gasless wallets, biggest ETH trend
- wagmi + viem + RainbowKit — modern frontend stack
- Layer 2 deep dive — Arbitrum, Base, zkSync
- Smart contract auditing — Slither, Mythril
- The Graph, Chainlink, RWA tokenization

**Solana**
- Token-2022 (Token Extensions) — new SPL standard
- Jupiter SDK — swap aggregation used in every DApp
- Helius webhooks + DAS API — standard indexing stack
- MEV + Jito bundles — highest-paying niche
- Yellowstone gRPC, @solana/web3.js v2
