Phase 6 — Projects (Build All 3)
1. DeFi Lending Protocol
Solidity · Hardhat/Foundry · wagmi · Next.js · The Graph
Users deposit ETH as collateral and borrow stablecoins. Includes liquidation logic, interest rate model, and a live dashboard showing on-chain positions via subgraph.
2. NFT Marketplace with Royalties
Solidity · ERC-721/2981 · RainbowKit · IPFS/Pinata · Next.js
List, buy, sell NFTs with on-chain royalties (ERC-2981). Metadata stored on IPFS. Includes a wallet-connected frontend with real-time event indexing.
3. Smart Wallet (Account Abstraction)
ERC-4337 · Solidity · Pimlico bundler · wagmi · Base L2
Build a gasless smart wallet using ERC-4337. Users sign transactions without needing ETH for gas (paymaster pays). Deploy on Base — most hirable project in 2026.

Anchor · Token-2022 · Metaplex · Next.js · Helius webhooks
